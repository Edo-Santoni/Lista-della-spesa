# Grocery List Simulator

This project is a simple **web application that simulates a grocery shopping list**.  
It allows users to:

- **Add items** to the list  
- **Delete a single item**  
- **Delete all items**  
- **Modify existing items**  

The goal is to provide a minimal, functional example of a CRUD-style interface using PHP and a local server environment.

---

## Features

- **Interactive grocery list** with dynamic updates  
- **Add / Edit / Remove** items easily  
- **Clear the entire list** with one action  
- Simple and lightweight structure suitable for beginners learning PHP, HTML, and basic backend logic

---

## Requirements

To run this project, you must host it on a server capable of executing PHP.  
You can use any server environment, but the simplest option is:

- **XAMPP (Apache + PHP)**  
  Personally, I run the project on my local machine using **XAMPP Apache**.

---

## How to Run the Project

1. Install **XAMPP**
2. Place the project folder inside the `htdocs` directory
3. Start **Apache** from the XAMPP control panel
4. Open your browser and navigate to:  
   `http://localhost/your-folder-name`
